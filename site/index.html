skrt

A React/MDX static site generator (SSG) with launch control

About

skrt is a command-line tool used to build static HTML websites with minimal effort. It's really well-suited for building documentation sites but is flexible enough to build other kinds of static sites like blogs or portfolios as well. It was heavily-inspired by Charge.

Features

  • Generate static HTML using React/MDX
  • Easy-to-use layout system using custom or built-in layouts
  • Smart metadata system (e.g. page title, description, slug) to simplify generating a complete website
  • Watch mode that automatically rebuilds/refreshes for development using Browsersync
  • And yes, it's fast, skrt! skrt!

Quick Start

  1. Install skrt:

    yarn add @generates/skrt --dev

  2. Create a docs directory with an index.mdx file containing the documentation index.

  3. Add the build command as a script to your package.json:

    "docs": "skrt build docs site"

    And run it with:

    yarn docs

    Or run it manually:

    yarn skrt build docs site

  4. During development, run it in watch mode:

    yarn docs -w

CLI Commands

build [source] [output]

Build the site from the [source] directory into the [output] directory.

Options

  • --watch, -w - Watch files in the [source] directory for changes and rebuild the site into the [output] directory

watch [source] [output]

Watch files in the [source] directory for changes and rebuild the site into the [output] directory. The site will be served using BrowserSync and your browser will automatically be refreshed after each build.

serve [output]

Serve the built site from the [output] directory.
